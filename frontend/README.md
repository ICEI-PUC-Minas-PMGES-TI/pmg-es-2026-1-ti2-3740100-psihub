# PsiHub - Frontend - Agendamento de consultas

Tela de agendamento de consultas psicológicas desenvolvida para o projeto PsiHub.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Seleção de psicólogos
- Calendário interativo
- Seleção de horários
- Bloqueio automático de datas passadas
- Bloqueio automático de horários já vencidos
- Timeline de progresso do agendamento
- Sidebar responsiva
- Resumo da consulta em tempo real
